# #37 Color Tap & Copy

A hyper-minimalist color picker for instant hex code retrieval. Tap or slide to pick, then copy with one touch.

### Core Philosophy
* **1 Tool / 1 Action:** Purely focused on color picking.
* **Performance:** Browser-only, ultra-lightweight, and mobile-optimized.
* **Privacy:** No login / No tracking / No ads.

### Features
* **Interactive Grid:** Tap or slide across the 240-color palette for real-time previews.
* **One-Tap Copy:** Instant clipboard copy with visual confirmation.
* **PWA Ready:** Designed to be "Add to Home Screen" for a full app experience.

### Design System
* **Background:** #fff / **Main:** #333 / **Accent:** #eee
* **UI:** 15px rounded corners on all elements.
* **Mobile Fixes:** Optimized for iOS Safari appearance and touch-action.
